Overview

Purpose

The purpose of this site is to provide a simple and efficient dice rolling tool for the tabletop roleplaying game Vampire: The Masquerade V5. The site allows users to input the number of dice and hunger dice, then simulate a roll using official game mechanics. It calculates successes, identifies critical successes, and detects messy criticals or bestial failures. This tool is intended to streamline gameplay and reduce reliance on physical dice.

Audience

The primary audience for this site is myself and my friends who are currently participating in a Vampire: The Masquerade Chronicle. It is designed for players and storytellers who want a fast and reliable way to roll dice and interpret results, especially during online or hybrid sessions.

Dynamic elements

JavaScript will be used to dynamically simulate dice rolls based on user input. Users will enter the number of total dice and hunger dice, and the script will generate random values for each die. The program will calculate successes (6 or higher), detect critical successes (pairs of 10s), and determine messy criticals or bestial failures depending on hunger dice results. The page updates instantly without reloading and displays results clearly.

Content

Home page

The home page will feature the main dice rolling tool. Users will input the number of dice and hunger dice, then press a button to roll. The results will display individual dice values, total successes, and whether the result is a critical success, messy critical, or bestial failure. The layout will be simple and centered for easy use during gameplay.

Dice Rules Page

This page will explain the mechanics of Vampire: The Masquerade V5 dice rolling. It will describe how successes are counted (6 or higher), how criticals are formed (pairs of 10s), and how hunger dice influence results such as messy criticals and bestial failures. This page helps new players understand the system.

Wireframes

Home

The home page will include input fields for total dice and hunger dice at the top, a roll button in the center, and a results section below. Results will display both individual dice and a summary of the outcome.

Dice Rules Page

The rules page will be text-focused with headings and sections explaining the dice system. It may include example rolls to demonstrate criticals and failures.
